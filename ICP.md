## 1. コアペルソナ（第一ペルソナ）: 港区キャバ嬢・ラウンジ嬢（20代）

> このペルソナは「よるのことば」の最初に共鳴する核。アプリのトーン・世界観はこの層から設計し、拡張ペルソナ（§1-B）に声を届ける。

### Demographics
- **年齢**: 20-29歳（中央値 23-24歳）
- **性別**: 女性 98%
- **居住**: 東京 23区内（特に港区・中央区・渋谷区・新宿区）
- **職業**: キャバ嬢・ラウンジ嬢・ガールズバー・高級会員制クラブ勤務
- **可処分所得**: 月 ¥300K-1.5M（変動大・請求制の店もあり）
- **勤務時間**: 20:00-04:00 が本番、朝帰り→日中就寝のサイクル

### Psychographics
- **情緒労働の蓄積**: 毎晩数十人の男性相手の接客で感情の消耗が激しい
- **スピリチュアル耐性高**: 占い師通い・パワーストーン・護符・開運グッズへの抵抗なし
- **共感と承認への渇望**: 「わかってほしい」「あなたは特別」のメッセージに強く反応
- **自己分析欲求**: 「私ってどういうタイプ？」「なんでこんなに疲れるんだろう」
- **仲間との共有文化**: 店LINEグループ・Instagram ストーリー・TikTok で結果を共有
- **変化と直感に敏感**: 「なんか胸騒ぎ」「今日ヤバい気がする」で行動を変える
- **"美しい"を大事にする**: 見た目の美しさは生活の一部、アプリのデザインでも妥協しない

### コアペルソナの1日のサイクルとタッチポイント
```
13:00 起床 →  スマホでインスタ → ぼーっと → 朝食
14:00 準備    美容院 / ネイル / ジム
17:00 出勤準備 メイク / 着替え → 同伴待ち
19:00 同伴開始
20:00 出勤
02:00 退勤     タクシー移動中にスマホ → ←【ここで接触】
03:00 帰宅    お風呂 → スマホで情報収集 → ←【ここで接触】
04:00 就寝前   占い・SNS・動画視聴 → ←【ゴールデンタイム】
13:00 翌日
```
**ゴールデンタイム（コアペルソナ）**: 午前 2:00-5:00 の「おつかれさま枠」。仕事終わりに自分をいたわる時間。これが「よるのことば」の名前の由来にもぴたりと符合。
なお「よるのことば」の「夜」は夢診断（夜に見る夢）にかけた言葉遊びであり、夜職専用サービスを意味しない。深夜・寝る前・夜の不安な時間に使うすべての人に開かれている。

- 「お客さん」「指名」「同伴」「アフター」

### NG 表現
- 固い／上から目線（「〜べきです」「〜してください」）
- 説教臭い（「努力が足りません」「もっと頑張って」）

## 1-B. 拡張ペルソナ（第二ペルソナ）: 占い・夢診断が好きな 10 代後半-20 代女性

> オーナー方針「幅広い F 層を狙いたい」に基づき 2026-05-17 追加。コアペルソナの声（姉貴分・深夜ラジオ）はそのまま維持し、聴き手の幅だけを広げる。

### Demographics
- 年齢: 16-29 歳（中央値 20-24 歳）
- 性別: 女性 95%
- 居住: 全国（地方含む）
- 職業: 学生・OL・パート・アルバイト。夜職に限定しない
- 可処分所得: 月 ¥30K-200K（学生は低め、OL は中程度）
- 生活リズム: 多様。23:00-2:00 に寝る前スマホ利用が共通

### Psychographics
- 占い・夢診断・タロット・星座をエンタメとして日常的に楽しむ
- 感情労働・人間関係・恋愛・将来不安を抱えている（夜職に限らない）
- 「なんかモヤモヤ」「この夢どういう意味？」と思ったらスマホを開く
- 自己理解・自己分析欲求が強い（MBTI・動物占い・エニアグラム経験者）
- SNS で占い結果をシェアする文化に親和性が高い
- 「かわいい」「刺さった」と感じたコンテンツを友達に回す
- スピリチュアルをガチで信じるというより「面白い・当たってる気がする」感覚での消費

### 利用文脈（コアペルソナ拡張・夜職前提を外した接触タイミング）
- 寝る前（23:00-1:00）にスマホを見ながら「今日の夢どんな意味だろ」
- 恋愛でうまくいかない夜に「これって何かのサイン？」
- 不安で眠れない夜に気持ちを落ち着けるため
- 朝起きて夢を覚えているうちに記録・診断したい
- 友達と「これ当たりすぎ」「やばい」と盛り上がる素材として

### 夜職インサイダー語を使わない語彙ガイド
この層には「同伴」「アフター」「指名」を使わない。共通語彙（§1 使う言葉の「共通」欄）とブランドトーンで届ける。

## 4. Key Insight（戦略統合）

コアペルソナの最大のペインは「夜中 3 時に誰にも相談できない感情を整理する手段がない」。インスタ占い師の DM は時間外、友達は寝てる、対面占いは通うのが面倒。

拡張ペルソナ（§1-B）に共通するペインは「深夜・就寝前の"なんかモヤモヤ"を受け止めてくれる場所がない」。夢の意味が知りたい、恋愛の答えが欲しい、感情を整理したい、という需要は夜職に限らず深夜に集中する。

よるのことば は「深夜のスマホの中の姉貴分」を占めるべきポジション。「3 時 AM のおつかれ枠」（コアペルソナ）も「寝る前の夢診断枠」（拡張ペルソナ）も、同じ声で届く。

### 5-1. 姉貴分の本音モード（2026-05-16 creative-director 識別性設計 判断A 物理化）

**定義**: 全員にやさしい「AI 推奨の励まし」を断り、「同じ夜にいる者どうしの対等さ」を言語化する温度。

**温度の軸: 平均値 vs よるのことば**

| 場面 | 平均値（断るべき） | よるのことば（目指す温度） |
|---|---|---|
| 深夜の挨拶 | 「今日のあなたは大丈夫。ゆっくり休んでね」 | 「まだ起きてるんだ。…まあ、わたしもだけど」 |
| 答えを求めてきたとき | 「きっとうまくいくよ」 | 「答えはもう自分でわかってるでしょ」 |
| 長い一日を終えたねぎらい | 「おつかれさまでした」 | 「おつかれ。今夜もよく帰ってきたね」 |

**語尾・温度の規定**

- 突き放しは「冷たさ」でなく「対等さ」として設計する。突き放し ≠ 説教
- 「答えはもう自分でわかってるでしょ」「まあ、わたしもだけど」のような共犯感のある本音が正解
- 「全員に届ける」ことを諦める（ユーモア能力: 朝・昼ユーザーには少し刺さらなくていい）
- deep モードは深夜2-4時に開いた人にしか届かない温度まで振り切る
- ただし「退勤後の帰宅」「今夜もよく帰ってきたね」等のコピーは、夜職の退勤だけでなく、長い一日を終えて夜にようやく自分の時間を持てたすべての人（拡張ペルソナ含む）に重なる。聴き手の幅は広いまま声は変えない

**絶対 NG（姉貴分モードでも禁止）**

- 「努力が足りない」式の説教（BRAND_RULES の NG 表現と同一）
- 「頑張れば大丈夫」「きっとうまくいく」（万人向けポジティブ感情過多 = スロップ）
- 過度な共感演技（「すごくわかるよ…」を3回繰り返すような過剰さ）

**識別性ゲート（本モードの評価基準）**

全コピーに「AI が合理的に推奨する平均値を、ここで断った」という1行が言語化できること。言語化できないコピーは平均値への後退として REVISE。
