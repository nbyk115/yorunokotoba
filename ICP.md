## 1. Primary ICP: 港区キャバ嬢・ラウンジ嬢（20代）

### Demographics
- **年齢**: 20-29歳（中央値 23-24歳）
- **性別**: 女性 98%
- **居住**: 東京 23区内（特に港区・中央区・渋谷区・新宿区）
- **職業**: キャバ嬢・ラウンジ嬢・ガールズバー・高級会員制クラブ勤務
- **可処分所得**: 月 ¥300K-1.5M（変動大・請求制の店もあり）
- **勤務時間**: 20:00-04:00 が本番、朝帰り→日中就寝のサイクル

### Psychographics
- **情緒労働の蓄積**: 毎晩数十人の男性相手の接客で感情の消耗が激しい
- **スピリチュアル耐性高**: 占い師通い・パワーストーン・護符・開運グッズへの抵抗なし
- **共感と承認への渇望**: 「わかってほしい」「あなたは特別」のメッセージに強く反応
- **自己分析欲求**: 「私ってどういうタイプ？」「なんでこんなに疲れるんだろう」
- **仲間との共有文化**: 店LINEグループ・Instagram ストーリー・TikTok で結果を共有
- **変化と直感に敏感**: 「なんか胸騒ぎ」「今日ヤバい気がする」で行動を変える
- **"美しい"を大事にする**: 見た目の美しさは生活の一部、アプリのデザインでも妥協しない

### Jobs-to-be-Done（JTBD）
| Job | 現状の解決策 | 支払い意思 |
|---|---|---|
| 感情を整理したい | 占い師に電話（¥5,000/回）、友達に愚痴 | 月 ¥2,000-10,000 |
| 自分のタイプを理解したい | MBTI・動物占い・インスタ占い師 | 月 ¥500-2,000 |
| 今日どう動くべきか知りたい | 星座占い（無料）・タロット・電話占い | 月 ¥1,000-5,000 |
| 仲間とシェアして盛り上がりたい | Instagram ストーリー・TikTok トレンド | 無料 |
| 恋愛・仕事のモヤモヤを整理したい | 占い師 1回 ¥5,000-20,000 | 月 ¥3,000-20,000 |

### 1日のサイクルとタッチポイント
```
13:00 起床 →  スマホでインスタ → ぼーっと → 朝食
14:00 準備    美容院 / ネイル / ジム
17:00 出勤準備 メイク / 着替え → 同伴待ち
19:00 同伴開始
20:00 出勤
02:00 退勤     タクシー移動中にスマホ → ←【ここで接触】
03:00 帰宅    お風呂 → スマホで情報収集 → ←【ここで接触】
04:00 就寝前   占い・SNS・動画視聴 → ←【ゴールデンタイム】
13:00 翌日
```
**ゴールデンタイム**: 午前 2:00-5:00 の「おつかれさま枠」。仕事終わりに自分をいたわる時間。これが「よるのことば」の名前の由来にもぴたりと符合。

## 2. Secondary ICP: ジェネラルスピリチュアル層

### Demographics
- 25-40歳 女性（70%）+ 男性（30%）
- 全国
- 可処分所得 月 ¥50K-200K
- OL・フリーランス・主婦・学生

### Psychographics
- 占い・タロット・星読みを日常的にチェック
- Co-Star / The Pattern ユーザー（海外アプリ経験者）
- Instagram の占い師フォロー経験あり
- 自己探求型、心理テスト好き
- MBTI・動物占い・エニアグラム等の性格診断ファン

### Use case
- 朝のルーティンで今日の運勢チェック
- 気になる相手との相性確認
- 自分のタイプを友達に共有
- モヤモヤしたときに占いで背中を押してもらう

## 3. 非ターゲット（やらないこと）

- 男性中心のガチ本格派タロット層（専門用語好き）
- 50代以上の電話占い師依存層（アプリに抵抗）
- 「占いは迷信」という合理主義層
- 無料のみ層（広告モデルで消耗しない）
- B2B案件

---

## 4. Key Insight（戦略統合）

Primary ICP の最大のペインは **「夜中 3 時に誰にも相談できない感情を整理する手段がない」**。
電話占いは高い、友達は寝てる、占い師は通うのが面倒。

よるのことば は「3 時 AM のスマホの中の占い師」を占めるべきポジション。
