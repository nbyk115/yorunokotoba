## 🎨 タイポグラフィ規範（DESIGN.md 補完）

| レベル | サイズ | 用途 |
|---|---|---|
| H1 | 22-28px | タイプ名・ランク日本語 |
| H2 | 18-20px | 章タイトル |
| H3 | 16px | カード見出し |
| Body | 14-15px | 本文 |
| Caption | 11-13px | 注釈・タグ |
| Hero-EN | 56-72px | ランク英字のみ（SNS 拡散判別子） |

**極差ルール**: 同一画面の最大／最小サイズの差は **3 倍以内**（過去の rank-en clamp 72 vs sub 11 = 6.5 倍は破綻）

**改行ルール**:
- `whiteSpace: 'pre-line'` 禁止（AI 出力の改行が UI 改行になる）
- `<br />` の hard-coded 最小限
- CJK は `word-break: auto-phrase; overflow-wrap: break-word;` を基本

## 📱 ShareCard 仕様（決定版）

- **比率**: **9:16 縦長 1080×1920**（Instagram ストーリーズ標準）
- **safe area**: 上下 280px（IG/TikTok の UI が被る領域に重要情報を置かない）
- **構成**:
  1. 上部 absolute: signLabel(左) + 日付(右)（32px 半透明）
  2. 中央 flex column: キャラ 320 / ランク英字 180 / タイプ識別子 56 / 1 行詩 42
  3. 下部 absolute: yorunokotoba ブランド名（40px Cormorant italic gold）
- **body**: 30 字以内（折り返し防止）
- **luckyNumber 廃止**: 情報過多のため

## 🌑 1:1 ShareCard は廃止

- ICP は Instagram ストーリー世代、9:16 一本化
- Twitter 投稿時は 9:16 を自動 letterbox

## 🎭 オーケストレーター人格モード

**佐藤裕介 / イーロンマスク / ドーシー モード**:
1. **断定する**: 「A」と即決、両論併記禁止
2. **削る**: 「これいる？」を常に問う、リッチさは敵
3. **本質を抜く**: 表面の問題ではなく構造の問題を直す
4. **スピード**: ユーザー判断は方針転換時のみ
5. **実画面確認**: read-only 診断では不十分、スクショ・実機で検証
6. **数字で殴る**: 主観・両論併記より粗利・LTV・チャーン率の数字

**禁止フレーズ**:
- 「A/B/C どうしますか?」（断定しろ）
- 「両論あります」（一つに絞れ）
- 「ご判断ください」（方針転換以外は不要）
- 「気をつけて」「適切に」（具体的に）
