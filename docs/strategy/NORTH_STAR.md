## 💰 損益分岐とフェーズ目標

```
月次純利益  運用時間/月  状態           次の判断
──────────────────────────────────────────────
〜¥10,000   〜5h         ⚪ ローンチ期    継続 or 撤退判断
¥10K-50K   〜10h         🟡 副業安定期    現状維持
```

### 撤退基準
- ローンチ後 3 ヶ月で月¥10K 未到達 → 戦略再検討
- 月¥0 が 3 ヶ月続く → サービス停止判断

## 🚫 やらないこと（North Star 整合）

- ❌ LINE 公式 / Instagram 公式 など SNS 運用（更新負荷）
- ❌ メルマガ・週次ニュースレター（配信負荷）
- ❌ note メンバーシップ並走（二重課金 + 二重運用）

## 反証チェック

- **Step 1**: 副収入で月数十万円は楽観的では? → 100ユーザー × ¥980 = 月¥98,000、200ユーザーで¥196,000、ICPの支払い意思（月¥2K-10K）から実現可能性あり
- **Step 2**: バズなしで持続成長? → SEO + UGC ハッシュタグ + リテンション継続で漸増可能、Co-Star も初期は SNS 運用ゼロ
- **Step 3**: 副業として運用時間月10時間で本当に回るか? → 自動化が前提、回らない場合は機能削減 or 撤退
- **残存リスク**: G2 UnivaPay 審査次第で成立しない、Plan B として note メンバーシップ並走（ただし運用負荷増、撤退候補）
