- **Status**: ✅ 確定（M1 ローンチ価格）

## 1. 確定価格

**Premium 単一プラン: ¥980/月（税込）**

- 課金プロバイダ: UnivaPay（`docs/payment/univapay-integration.md` 参照）
- 課金サイクル: 月額自動継続（解約は次回更新日前まで有効）
- 解約: マイページから1クリック（G8 で実装予定）

## 2. 採用根拠（5軸）

### 2.1 創業者直感（最強の市場シグナル）
創業者自身が ¥1,980 を「払わない」と感じる価格は伝道できない。マーケコピー・SNS 発信・営業トークすべての説得力に直結する。

### 2.2 PMF 検証フェーズ最適化
M1 は ARR より「継続率・NPS・口コミ係数 K」の検証フェーズ。¥1,980 は契約後の認知不協和→チャーン誘発リスクが高い。初期トラクション > 単価最適化。

### 2.3 競合ポジショニング
| 競合 | 月額 | 差分 |
|---|---|---|
| Co-Star Premium | ¥1,800 相当 | -¥820（45% off） |
| CHANI Pro | ¥1,200 相当 | -¥220（18% off） |
| LINE 占い（公式） | ¥500-1,500 | 中間帯 |
| note メンバーシップ（占い系平均）| ¥500-1,000 | 同等 |
| インスタ占い師 DM 鑑定 | ¥3,000-10,000（都度）| 月 1 回未満で割高 |

→ CHANI/Co-Star より明確に下、note 系より上の「**プロ品質 × 日本市場フィット**」ポジション.
**アンカリング戦略**: UI 上で他サービスとの価格比較は出さない（ドーシー流シンプル化）.
「月¥980（税込）· 自動更新 · いつでも解約できるよ」のみで訴求.

### 2.4 ICP 支払い能力との整合
（`ICP.md` Section 1 参照）

| ICP セグメント | JTBD 上限 | ¥980 適合 |
|---|---|---|
| Primary（港区夜職）| 月 ¥2,000-10,000 | ✅ 大幅余裕 |
| Secondary（OL/フリーランス）| 月 ¥500-2,000 | ✅ レンジ内 |

→ ¥980 は **両 ICP の支払い能力範囲内**。¥1,980 は Secondary を実質的に切り捨てる。

### 2.5 PL 試算（佐藤 PL 思考）
```
ARPU:           ¥980/月
UnivaPay 手数料: 3.2%（¥31）
粗利:           ¥948/月（粗利率 96.8%）
LTV（6ヶ月想定）: ¥5,880 → インスタ占い師 DM 鑑定 1 回分相当
LTV（12ヶ月想定）: ¥11,760
CAC 上限（LTV/3 ルール）: ¥1,960（6ヶ月） / ¥3,920（12ヶ月）
ブレイクイーブン: 月間 Premium 100 ユーザー（固定費 ¥95K想定 → 粗利 ¥94.8K）
```

## 3. 棄却した代替案

| 案 | 棄却理由 |
|---|---|
| **¥480/月** | 粗利インパクト弱（¥464/月）・"安かろう"ブランド毀損・ICP の支払い能力を取り逃す |
| **¥1,480/月** | 創業者納得感不足。M3 で値上げ判断する方が検証コスト低い |
| **¥1,980/月** | 創業者直感不一致・初期摩擦過大・Secondary ICP 切り捨て |
| **2段階並列（¥980/¥1,980）** | 初期は「選択疲れ」が CVR を下げる。LTV データ蓄積後の M6 で十分 |
| **フリーミアム + ¥980** | 既に基本機能無料 + Premium ¥980 = 実質フリーミアム。追加層は不要 |

## 4. 段階引き上げロードマップ

```
M1（今月）: ¥980 で βローンチ
M2:        チャーン率・LTV・初週リテンション計測
M3:        ¥1,480 への A/Bテスト判断
           - 既存ユーザー: ¥980 据え置き（grandfathering）
           - 新規ユーザー: 50/50 で ¥980 vs ¥1,480
           - 判定基準: CVR 低下 < 30% AND ARPU 上昇 > 30%
M6:        Premium Plus ¥1,980 検討
           - 差別化: 占い師1on1（月1回）・カスタム鑑定書PDF・優先サポート
```

## 5. KPI 監視設計（M2 client-success / kpi-analytics 連携）

| KPI | 目標値 | 警戒ライン | 計測 |
|---|---|---|---|
| 初週リテンション | > 60% | < 40% | Firestore subscription.status 日次集計 |
| M1 チャーン率 | < 10% | > 20% | 解約イベント / 月初 active |
| LTV（90日推定）| > ¥2,900 | < ¥1,500 | ARPU × 平均継続月数 |
| 解約理由 Top3 | "高すぎる" 比率 < 20% | > 35% | 解約 UI のアンケート |

→ "高すぎる" > 35% で価格再検討。"内容不足" > 35% でプロダクト改善優先。

## 6. 反証チェック結果

✅ **Step 1（自己反証）**:
- 「¥1,980 で取れる層を捨てるのでは？」 → M6 Premium Plus でアップセル回収可能。初期 PMF 検証優先
- 「¥980 は安すぎて品質を疑われる？」 → CHANI ¥1,200 より下だがブランド・コンテンツ深さで差別化
- 「価格弾力性データなしで決めて良いか？」 → 自社データなく、競合データ + ICP データで判断。M3 で実測 A/Bテスト

✅ **Step 2（構造反証）**:
- LTV 計算の継続月数仮定（6ヶ月）の根拠 → 占いアプリ業界平均 4-8 ヶ月、中央値採用
- 粗利率 96.8% の根拠 → UnivaPay 公表手数料 3.2%（占い・スピリチュアル業種）
- 競合価格データ → 各社公式サイト 2026-04 時点

✅ **Step 3（実用反証）**:
- 値上げ時の既存離反 → grandfathering で回避
- ¥980 で来た層がそもそも継続率低い可能性 → M2 でリテンション細分析必須
- UnivaPay 障害時の代替 → note メンバーシップ ¥980 並走で冗長化（S2 検討）

🔺 **残存リスク**:
- 創業者直感に依存した判断（データ薄）→ M3 までに実測で検証
- "高すぎる" 解約理由 > 35% の場合は ¥780 への引き下げを検討（grandfathering 不要）
- 競合（CHANI/Co-Star）の日本市場本格参入で価格優位性が消える可能性 → M6 で再評価
