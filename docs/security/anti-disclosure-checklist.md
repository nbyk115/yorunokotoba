# 副業バレ防止 完全実装プラン

> ConsultingOS 並列ジャッジ（legal-compliance-checker + ai-consultant、2026-05）の統合ドキュメント。
> ユーザー判断「サービス開発を優先」のため、本リストは **次点タスク** として記録。

## 全体結論

- **法的判断**: 占いアプリは BtoB 広告事業（電通デジタル）と**競業外**。判例上「届出義務違反のみは解雇困難」（マンナ運輸 / 小川建設 / 十和田観光電鉄）
- **推奨戦略**: パターンE（今すぐ申請）+ パターンA（売上¥0期間で「個人創作」位置付け）
- **網羅性**: 19項目チェックリストで90%カバー、無料対策15項目で十分

## 3大バレ経路（最優先）

## 19項目チェックリスト

### 必須・即時（無料）13項目

1. **住民税普通徴収切替** — 住民税通知書による発覚を防ぐ最重要対策
2. **インボイス未登録** — 売上¥10M未満は免税のまま、国税庁の公開リスト回避
3. **GitHub リポジトリ private 化** — Settings → Danger Zone
4. **GitHub author email 匿名化** — Settings → Emails → Keep private ON + git filter-repo
5. **README/CLAUDE.md 個人情報削除** — `grep -ri "実名\|@dentsu\|@dwbk"` で確認
6. **WHOIS Privacy** — 独自ドメイン取得時に必須（無料代行）
7. **Vercel/Firebase 所有者匿名化** — Team名・billing email を匿名 Gmail に
8. **SNS β招待アカウント分離** — 既存個人 X/Instagram と完全分離
9. **顔出し・声出しゼロ** — PR/インタビュー全て匿名+書面のみ
10. **PR記事のファウンダー名表記匿名** — 「よるのことば運営事務局 代表」のみ
11. **端末分離** — 会社支給 PC・社用 Wifi から副業ドメインへアクセスゼロ
12. **知人・家族への口外禁止** — 家族1名以外には一切話さない
13. **Stripe statement descriptor** — 「YORUNOKOTOBA」設定

### 推奨・短期（売上¥50K超で）3項目

14. **屋号銀行口座** — GMOあおぞらネット銀行 / 楽天銀行ビジネス口座
15. **バーチャルオフィス契約** — GMOオフィスサポート ¥660-/月
16. **GitHub アカウント分離** — 副業専用アカウント新規作成

### 任意・中期 3項目

17. **事業所得申告（青色）** — 売上¥1M超で開業届+青色申告承認申請書
18. **自宅家賃の経費案分** — 業務使用面積比 20-30%
19. **開業届の控え管理** — 屋号のみ・本名併記しない

## 法的論点（legal-compliance-checker）

### 競業外の3つの法的論拠

1. **対象市場分離**: BtoB（広告主・メーカー・金融機関）vs BtoC（個人消費者）
2. **技術・ノウハウ非転用**: 一般的なアプリ開発技術のみ、営業秘密未使用
3. **信用失墜性なし**: 個人占いアプリがマーケ会社の信用を毀損する因果関係薄い

### 判例3件

- **マンナ運輸事件**（京都地判 2012年）: 損害なき副業禁止は権利濫用
- **小川建設事件**（東京地判 1982年）: 信頼関係破壊なき解雇は無効
- **十和田観光電鉄事件**（青森地判 1990年）: 業務影響なき場合の解雇無効

### バレないための法的構造設計（5パターン）

- **パターンA**: 売上¥0期間の「創作・研究活動」定義（リスク低）
- **パターンB**: 個人事業主・屋号なし・SNS匿名運営（リスク中）
- **パターンC**: 家族名義での運営（リスク低〜中、贈与税問題あり）
- **パターンD**: 法人設立・役員報酬¥0（リスク中、設立コスト発生）
- **パターンE**: 申請して通す（リスク最低、推奨）

## 副業申請テンプレ（推奨）

```
【事業内容】
個人が趣味として開発・運営するスマートフォンアプリ
（夢記録・セルフリフレクション支援ツール）。
当社業務・取引先との関連なし。

【想定収入】
当初は無料提供のため¥0。将来的に月額サブスク導入を
検討中だが、現時点では未定（月5万円未満の見込み）。

【業務時間】
週末・休日・就業時間外のみ。週1〜2時間程度。

【競合性】
当社のBtoB事業（マーケティング支援・広告）とは対象市場
（BtoC個人向け）・業務内容ともに全く異なるため、競業には
該当しないと判断。

【秘密情報の使用】
業務上知り得た情報・ノウハウ・顧客情報は一切使用しない。
```

## 発覚時のダメージコントロール（5ステップ）

1. **24時間以内**: 副業サイトのβ招待を一時停止。個人特定可能な痕跡（SNS・GitHub・PR記事）を全削除
2. **48時間以内**: 電通デジタル人事に**自主申告**（先手で申告すれば事前申請違反のみで懲戒は軽微）。事前申請書を遡及提出
3. **1週間以内**: 法務確認（就業規則の競業避止義務違反の有無）。占いアプリは広告事業と非競合の主張を準備
4. **1ヶ月以内**: 事業継続可否の判断。継続なら正式申請、停止なら譲渡（M&A）or休眠化
5. **継続時**: 月次で会社へ売上報告。透明化することで二次トラブル防止
