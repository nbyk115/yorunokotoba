# コンテンツ展開計画 & カレンダー

## 結論
夜語コード体系を軸に、フロー型（SNS毎日投稿）とストック型（ブログ・キャラ図鑑）を統合したコンテンツ戦略を展開する。初月で24キャラ紹介を完走し、2ヶ月目以降は相性・ランキング・参加型コンテンツで継続的なエンゲージメントを構築する。

---

## 1. コンテンツ戦略の全体像

### 目的
| 指標 | 目標（3ヶ月後） | 計測方法 |
|---|---|---|
| アプリ利用者数 | 月間アクティブ +200% | Firebase Analytics |
| SNSフォロワー | X: 5,000 / Instagram: 3,000 | 各プラットフォーム |
| シェア率 | 占い結果の15%がシェアされる | シェアボタンクリック率 |
| 夜語コード認知 | 「#夜語コード」投稿数 500件/月 | ハッシュタグ計測 |

### ターゲット
- **コア**: 20-30代女性、占い・自己分析好き、MBTI経験者
- **拡張**: 10代後半-20代前半、TikTok/Instagram中心、推し活文化

### チャネル構成

| チャネル | 役割 | 頻度 | コンテンツ型 |
|---|---|---|---|
| X（旧Twitter） | 認知拡大・バイラル | 毎日2-3投稿 | フロー |
| Instagram | ビジュアル訴求・保存促進 | 毎日1投稿 | ストック寄りフロー |
| TikTok | 短尺動画・新規獲得 | 週3-4本 | フロー |
| アプリ内ブログ | SEO・深掘りコンテンツ | 週2本 | ストック |
| LINE/メルマガ | リテンション | 週1回 | ナーチャリング |

## 2. コンテンツシリーズ一覧

### シリーズA: キャラクター紹介（24日間連続）

**目的**: 全キャラを認知させ、「推しキャラ」文化の土台を作る
**期間**: 初月の24日間、毎日1キャラずつ公開

**投稿フォーマット（X / Instagram）:**
```
今日の夜語キャラクター #01

ゆめのこぶた [YM-KO]
レアリティ: N

「好き」の気持ちに正直すぎて、
いつも少しだけ損をする。

▶ 天真爛漫な愛されキャラ
▶ ベストマッチ: ほしのクマ
▶ あなたは何タイプ？ → [リンク]

#夜語コード #YMKO #よるのことば
```

**公開順序（バイラル最適化）:**
| 日 | キャラ | レアリティ | 公開理由 |
|---|---|---|---|
| Day 1 | ゆめのこぶた | N | 親しみやすさで入口に |
| Day 2 | まよなかのネコ | SR | 猫系で注目を集める |
| Day 3 | さくらのうさぎ | R | かわいい系で層を広げる |
| Day 4 | ひのトラ | R | ギャップで男性層にもリーチ |
| Day 5 | そらのユニコーン | SSR | 初SSRで話題性を作る |
| Day 6 | はなのパンダ | N | 癒し系で休息日 |
| Day 7 | つきのヒョウ | SR | 週末にクールキャラ投入 |
| Day 8-12 | 中堅キャラ5体 | N/R | 安定的に認知拡大 |
| Day 13 | かぜのオオカミ | SSR | 中盤のカンフル剤 |
| Day 14-18 | 残りの中堅5体 | N/R | 推し投票への布石 |
| Day 19 | ほのおのフェニックス | SSR | 終盤に向けて盛り上げ |
| Day 20-23 | 残りキャラ4体 | N/R/SR | ラスト前の伏線回収 |
| Day 24 | にじいろのドラゴン | SSR | 最終日にSSRで締め |

### シリーズB: 今週の夜語タイプ（毎週月曜配信）

**目的**: 定期コンテンツで継続的な訪問動機を作る
**開始**: キャラ紹介完走後（2ヶ月目〜）

**投稿フォーマット:**
```
今週の夜語タイプ占い [3/23-3/29]

▶ 絶好調: HO-TR（ひのトラ）
  → 今週は行動した者勝ち。迷ったらGO

▶ 要注意: YK-HC（ゆきの白鳥）
  → 完璧を求めすぎない週。60点でOK

▶ ラッキーペア: TK-HY × YM-KO
  → この二人が一緒にいると運気UP

あなたの今週は？ → [リンク]
#夜語コード #今週の運勢
```

### シリーズC: 夜語コード相性診断（毎週水曜配信）

**目的**: 二人組でのシェアを促進し、バイラルループを作る

**投稿フォーマット:**
```

TK-HY × YM-KO
つきのヒョウ × ゆめのこぶた

#夜語コード #夜語相性
```

### シリーズD: キャラクターの「裏の顔」（毎週金曜配信）

```
#夜語コード #SRUN #そらのユニコーン
```

### ファネル別KPI

| 認知 | インプレッション | 月100万 | X Analytics / Instagram Insights |

### A/Bテスト項目
- シェア画像のデザイン（コード表示あり vs なし）
- キャッチフレーズの表示位置（結果画面上部 vs 下部）
- CTA文言（「友達に教える」vs「あなたの夜語コードをシェア」）
