# 夜語コード体系 & キャラクター記号化戦略

## 目的
「よるのことば」の24キャラクターにMBTI的な「記号化力」を付与し、SNSプロフィールや会話で自然に使われる文化を作る。

---

## 1. 夜語コード体系

### 設計原則
- **2文字の属性コード + 2文字の動物コード = 4文字**
- MBTIの「INTJ」「ENFP」のように、4文字で個性が伝わる
- 規則性があり、体系を理解すれば初見でも意味が読み取れる
- SNSプロフィールに `#夜語コード:YM-KO` のように書きたくなる形式

### コード構造

```
[属性2文字] - [動物2文字]

属性コード（自然現象 → 2文字の略称）
動物コード（動物名 → カタカナ2文字の略称）
```
