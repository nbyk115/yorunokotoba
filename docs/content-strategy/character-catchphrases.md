# キャラクター別キャッチフレーズ集

## 設計方針
- 「あ、私これだ」と自己投影できる1行
- シェア画像・SNSプロフィールに映える長さ（20-30文字程度）
- キャラクターの本質的な魅力と「刺さる弱さ」を両立
- 占い結果画面・カード・OGP画像に表示する想定

## キャッチフレーズの使用箇所

### 占い結果画面
- キャラクター名の下に1行で表示
- フォント: Zen Maru Gothic 400, 14px
- カラー: var(--t2)

### シェア画像（OGP / Instagram / X）
- キャラクターアバターの下に配置
- 白抜き文字 on 半透明背景
- 最大2行で折り返し

### SNSプロフィール用テンプレート
```
夜語コード: [コード] | [キャッチフレーズ]
例: 夜語コード: TK-HY | 感情に溺れない。でも、感情がないわけじゃない。
```

### キャラクターカード（コレクション画面）
- カード裏面に表示
- タップで表示するイースターエッグ的演出

## キャッチフレーズの設計パターン

全フレーズに共通する構造的特徴:

1. **「でも」構文**: 表の印象 + 裏の本音（例: 完璧に見える人ほど〜）
2. **自己肯定構文**: ネガティブに見える特性 → 実はポジティブ（例: マイペースは〜強さ）
3. **問いかけ構文**: 読者に考えさせる（例: 常識を壊すのが怖い？）
4. **「〜ない」二重否定**: 誤解を解く（例: 何も言わないのは〜ないからじゃない）

これらのパターンにより、「自分のことを言い当てられた」感覚を生み、シェア率を高める。
