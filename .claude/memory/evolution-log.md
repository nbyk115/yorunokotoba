# Evolution Log — スキル進化記録

## ⚠️ 重要: 追記専用

**このログの実データは `evolution-log.jsonl` に保存される。**
JSONL形式（1行1エントリのJSON）は追記のみで過去の履歴が絶対に上書きされない。
本ファイル（.md）は人間向けの概要サマリーのみ。詳細は`.jsonl`を参照。

### 書き込みルール
1. 新しいエントリは `.jsonl` に**追記のみ**（既存行の変更禁止）
2. 1行1エントリ。各行は完結したJSON
3. 必須フィールド: id / date / target / type / title / rationale / result
4. ロールバック時も既存行を消さず、新しい「rollback」エントリを追記する

## 概要
スキルファイル・エージェントの全進化履歴を記録するログ。
変更の透明性を確保し、ロールバック判断の根拠を提供する。
