# Monetization Playbook — ConsultingOS副業収益化

> AIエージェントを使って副業収益を生むための完全ガイド。
> 全エージェントはこのスキルを参照し、アウトプットを「売れる形」に設計する。

---

## §1 収益化の基本原則

### 1.1 副業収益化の3ステップ
```
Step 1: 今すぐ売れるサービスをパッケージ化
Step 2: ConsultingOSでアウトプットを生成
Step 3: クライアントに納品 → 実績を積む → 単価を上げる
```

### 1.2 ConsultingOSが強い理由
- AIエージェントで「コンサルファーム品質」のアウトプットを1人で出せる
- 調査→分析→提案書→コンテンツの全工程を担える
- 納品速度が従来の1/3〜1/5（差別化ポイント）

### 1.3 価格設定の原則
- **コスト積み上げ禁止**: 時給ではなく「価値」で設定する
- **競合より20〜30%高く**: 品質で差別化するため安売り禁止
- **パッケージ販売**: 単発より継続契約を優先（LTV最大化）

## §3 チャネル戦略

> **ランサーズ・ここなら 非推奨**: 手数料20〜25%を取られ、価格競争に巻き込まれ単価が下がる。
> ConsultingOSの強みは「速さ×品質の非対称性」。これを正しく評価してもらうには**価値で選ばれるチャネル**が必須。

### 3.1 知人・OB/OGネットワーク【最速・最高成約率・今日から】
```
なぜ最強か:
- 信頼コストゼロ（すでに関係がある）
- 競合比較されない（指名で来る）
- 紹介連鎖で案件が雪だるま式に増える

今日やること:
1. 元同僚・上司・友人に「AIエージェントで調査・提案書を作れるようになった」と話す
2. 「社内の課題で試してみない？ 初回は実費だけで」
3. 納品後に「他に困ってる人いたら紹介してほしい」と必ず依頼する

目標: 最初の3件は紹介で埋める
単価: 初回3〜8万（実績作り）→ 紹介2件目から10〜20万
```

### 3.2 X（Twitter）【中期・インバウンド・高単価 DM】
```
戦略: 「見込み客が自分から来る」状態を作る

発信テーマ（週3回）:
- ConsultingOSの出力実例（競合調査・AI導入レポートのサマリー画像）
- 「〇〇業界の競合構造をAIで分析してみた→驚いた3つの発見」
- クライアント課題→解決→結果のストーリー（数字含む）
- AI活用の裏側プロセス公開（信頼獲得）

DM設計:
- プロフィール: 「AI戦略支援 / 競合調査・提案書を3日で納品 / DM受付中」
- 固定ポスト: 実績レポートの一部画像 + サービス概要 + 問い合わせリンク
- DM返信: 「30分ヒアリングさせてください」で即クロージング

単価帯: 5〜20万円/件（価格を明示すると高単価案件が来やすい）
目標: 3ヶ月で月1〜2件インバウンド
```

### 3.3 LinkedIn【法人・高単価・継続契約】
```
ターゲット:
- 中小企業経営者（従業員50〜500名）
- 事業会社のマーケマネージャー・経営企画
- スタートアップCFO・CMO

単価帯: 10〜50万円/件 / 顧問 5〜20万/月

アプローチ:
1. 専門記事投稿（週1〜2回）
   「AI×コンサルティング」専門家ポジション。数値付き業界分析を載せる
2. ターゲットへのDM
   相手の投稿に価値ある返信 → つながり申請 → 3往復後に「10分だけお話しできますか」
3. 単発 → 月次レポート契約（3〜8万/月）→ 顧問契約（10〜20万/月）に転換

目標: 6ヶ月で月1〜2件の法人案件
```

### 3.4 エージェント・マッチングサービス【すぐ稼げる・競合少】
```
推奨（ランサーズ/ここなら より単価が高い）:

① ビズリーチ（副業・フリーランス）
  - IT系・コンサル系の副業案件が豊富
  - 時給換算5,000〜15,000円の案件が存在
  - 登録無料・エージェントが案件を紹介

② Workship / Offers
  - スタートアップ向け副業マッチング
  - マーケ・戦略・分析系が多い
  - 週5〜10時間から受けられる案件あり

③ タイムチケット / MENTA（高単価設定で）
  - 1時間5,000〜20,000円のコンサル時間を販売
  - 「AIエージェント × 経営課題相談」で希少性を出す

④ コールドDM / コールドメール（法人直接）
  - LinkedInで「AI導入を検討中の企業」に直接アプローチ
  - テンプレ: 「御社の〇〇課題向けにAI活用の無料診断（30分）を提供しています」
```

### 3.5 自社LP + note【資産型・長期的インバウンド】
```
LP構成（3スクロール以内）:
- Hero: 「AIエージェントで競合調査を3日・5万円で」（数字+期間+金額）
- 実績: 「過去XX件 / 平均満足度X.X / 事例紹介」
- サービス: 3プランで選ばせる + 申込みボタン

note戦略（コンテンツファネル）:
- 無料記事: 「〇〇業界の競合調査レポート（要約版）」を定期公開
- 有料記事: 「フルレポート版」1,500〜5,000円
  → 有料記事購入者に「個別相談無料」でクロージング
- noteの読者 → LP → 問い合わせ のファネル

SEOキーワード（狙いやすい）:
- 「競合調査 外注」「AI導入 コンサル 個人」「提案書 代行 フリーランス」
```

## §7 アウトプット品質基準（クライアント向け）

全クライアント向けアウトプットは以下を必須とする:

1. **humanizer-ja適用**: AI臭を除去し「人間が書いた文章」に仕上げる
2. **エグゼクティブサマリー必須**: 冒頭1ページで結論・根拠・アクションを完結させる
3. **数値化**: 全主張に数値・出典を付ける（「大幅に」禁止）
4. **次のアクション明示**: 「で、何をすればいいか」を3つ以内で明示して終わる
5. **ビジュアル**: 表・図・フローチャートを最低1つ含める

## §8 反証チェック（受注前確認）

```
【このサービスを売れるか？チェック】
□ 「誰の・何の問題を解決するか」1文で言えるか
□ 「なぜConsultingOSでないとダメか」の差別化が説明できるか
□ 「価格に見合う価値があるか」クライアント目線で検証したか
□ 「納品できないリスク」はないか（スコープ過大・技術的制約）
□ 「続けてもらえるか」継続性の設計があるか
```
