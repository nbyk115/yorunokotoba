# Astroturfing 認識 + Counter-Detection 統合 (2026-05-15)

`social-media-strategist` + `pr-communications` + `marketing-director` + `legal-compliance-checker` + `brand-guardian` + `market-researcher` + `competitive-analyst` 連携の Astroturfing 拒否 + counter-detection フレーム。「やる」でなく「認識 + 警戒 + 判別」する立場。

出典: Joe Lim (Floodify) + Lane Brown 報道 (INFERENCE: ユーザー提示テキスト経由)。Floodify = 3 年運用、ピーク 65,000 偽アカウント、$1/1000 views 報酬。Justin Bieber Coachella 公演で paid clipper campaign 実例 (2026-04)。

## 1. 核心認識 (FACT、Joe Lim 発言)

- 「インターネット上の 90% は変装した広告」(Joe Lim 推定 INFERENCE)
- 「今やみんながこれをやっている。やってないなら、あなたは遅れている」(Joe Lim、astroturf 化の業界共通認識)
- 適用範囲: 音楽 + エンターテイメント + 政治 + 消費者製品 + セレブゴシップ + 「人々が他人の意見を知るためのあらゆるアプリ」

= 「世論が形成され、測定され、操作されるすべてが同時に起こっており、プラットフォームが生成するあらゆるシグナル (トレンド曲 / 反発 / トーキングポイント / みんなが突然同じこと言ってる感覚) は隠れた意図を持つ見えないアクターによって捏造可能」(Lane Brown)。

## 4. Counter-Detection: 真のシグナル vs 捏造シグナル判別

ConsultingOS が競合 / 投資先評価で「astroturf 化リスク」検知する 5 軸:

| 検知軸 | 真のシグナル | 捏造シグナルの特徴 |
|---|---|---|
| 1 アカウント年齢分布 | 多様 (新規 + 既存混在) | 短期間に大量新規アカウントから言及 |
| 2 トーン同質性 | 多様な表現 / 視点 | 同じフレーズ / トーキングポイント反復 |
| 3 言及時期集中 | 自然分散 | 短時間に集中 (paid campaign 期間と一致) |
| 4 エンゲージメント比率 | view ≈ like / share の自然比 | view 偏高 (paid view 買い) |
| 5 関連アカウント連鎖 | 多様な業界 / 地域 | 互いに follow / engagement する閉鎖クラスタ |

ConsultingOS 整合:
- xAI X アルゴリズム公開 (PR #172): ソースコード根拠で真のシグナル判別可能性増
- aeo-playbook §9 Lauren Price (PR #147): 真のブランドサーチ = AI 引用最強相関、捏造シグナルとの区別

## 8. アンチパターン 5 件

YOU MUST: 以下を検知したら即停止 + 案件撤退 / 設計やり直し:

1. 「みんなやってるから」論理 (Unkind Truth 違反)
2. 偽アカウント運用提案 (Hard Rule 17 + 各国法令違反)
3. paid promotion 開示なし (FTC / 景品表示法違反)
4. 短期 view 数増加のみ目標化 (小野寺流 CPC/CPA 主指標化禁止と類縁)
5. プラットフォーム規約違反自動化 (LinkedIn / X / Instagram 等)
