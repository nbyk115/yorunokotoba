# 売れる言葉スパイス辞典 + 6 心理カテゴリフレーム (2026-05-15)

`content-strategist` + `proposal-writer` + `sales-deck-designer` + `marketing-director` + `pr-communications` 連携のコピー設計フレーム。「AI にコピーを書かせる前に、まず言葉の材料を増やす」原則。

出典: kawai-word.pages.dev (INFERENCE: ユーザー提示テキスト経由) + Mr. かわい (推定) 「売れる言葉 150 + タイトル例 150」スパイス辞典。

## 1. 核心原則: コピー材料を増やす > AI に書かせる

「コピーが弱い人は語彙が少ないだけ」= 出力品質 = 入力素材品質に支配される (DeCourcy「ブリーフは第一のクリエイティブ行為」PR #148 と同型構造)。

NG: 「すごい」「便利」「おすすめ」のみ
OK: 6 心理カテゴリの語彙バリエーション

## 2. 6 心理カテゴリ (FACT、スパイス辞典分類)

| カテゴリ | 用途 | 例 (INFERENCE、辞典推定) |
|---|---|---|
| 好奇心 | 注意喚起、興味誘発 | 「実は...」「あの○○が」「○○の真実」 |
| 共感 | 読者の感情と接続 | 「あなたも？」「気持ちわかる」「○○な人へ」 |
| 権威 | 信頼性 / 専門性訴求 | 「○○大学」「業界第 1 位」「○年実績」 |
| 希少性 | 限定感 / 緊急性 | 「○名限定」「今だけ」「残り○個」 |
| 行動喚起 | 次のアクション促進 | 「今すぐ」「無料」「申込はこちら」 |
| 失敗回避 | 損失回避バイアス | 「失敗しない○○」「○○のリスク」「知らないと損」 |
