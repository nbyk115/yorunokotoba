# ビジュアル品質ルーブリック (creative-playbook §2.5 分離、2026-05-15 PR #216 物理化)

> creative-director 品質診断 (2026-05-15) を受けた数値ゲート。「ダサい/良い」の主観判定を採点可能基準に置換し、creative-director 最終レビューを「ルーブリック採点 → 80 点未満 REJECT」化する。examples で v1-v9 と 9 イテレーション回しても収束しなかった根本原因 = 採点根拠の主観依存、を構造解消。

## 採点 15 項目 (各 PASS = 1 点、合計 15 点満点、12 点 = 80% 未満は REJECT)

### タイポグラフィ (5 項目)

1. 本文行長 35-45 字 (日本語)。長すぎ = 視線移動過大、短すぎ = リズム断絶
2. 行間 line-height 1.7-1.8 (本文)
3. 最小フォントサイズ: 本文 14px 以上 / 注釈 12px 以上
4. H1 対 本文のサイズ比 2.0 以上 (視覚的階層の明確化)
5. フォント種別 2 種以内 (見出し + 本文、欧文/数値の Inter は別カウント可)

### カラー (3 項目)

6. 色数 3 色以内 (プライマリ + セカンダリ + アクセント、ニュートラルグレースケールは別)
7. コントラスト比 4.5:1 以上 (本文テキスト、WCAG 2.1 AA)
8. セマンティックカラー参照 (hex 直書き禁止、トークン経由)

### レイアウト・余白 (4 項目)

9. セクション間余白 48px 以上
10. 余白が 4px 基準スケール (4/8/12/16/24/32/48/64/96) に乗っている
11. 1 画面 (1 セクション) 1 メッセージ。複数主張の混在禁止
12. 視覚的階層が明確 (名前/見出し/本文のサイズが上から下へ流れる)

### 実装・配信 (3 項目)

13. レスポンシブ: @media ブレークポイント実装済 (モバイル max-width 768px)
14. OGP / Twitter Card meta 設定済 (共有時の表示品質)
15. DESIGN.md トークン準拠 (色・フォント・余白が DESIGN.md 定義値と一致、独自定義なし)

## 採点運用

- creative-director の最終レビューを「15 項目採点 → 12 点未満 (80% 未満) は REJECT 差し戻し」に変更
- 機械検証可能な項目 (6 色数 / 7 コントラスト / 13 @media / 14 OGP / 15 トークン準拠) は brand-guardian + template-injection-check.sh hook で自動採点
- 定性寄りの項目 (1 行長 / 4 階層 / 11 メッセージ) は creative-director 目視採点
- 採点結果を成果物に添付 (取得点/15 点 + REJECT 項目リスト)

## Hard Rule 13 対応 (追加 = 削除 1 セット)

本ルーブリック新設に伴い、ux-designer / creative-director の定性的「必須ゲート」のうち本ルーブリックと重複する項目は creative-playbook §2.5 / 各 agent description で削減対象。重複削減は次 PR で実施 (本 PR は references 新規 + §2.5 参照リンクのみ)。
