# Brand Distinctiveness: AI 時代の「断る力」と反スロップ (2026-05-15 PR #226 物理化)

> creative-playbook の識別性ゲート詳細。creative-director 判定 (2026-05-15) で物理化。本 references は creative-playbook §2.5 から参照。

## 1. スロップ = ゆるやかな死

AI で誰でも一定品質のアウトプットが手に入る時代、深刻な問題は「品質不良」でなく「均質化」。

研究 FACT (出典: WIRED 紹介、Imperial College London / Stanford / Internet Archive 共同研究、2026):
- 2022 年以降の新規ウェブサイト約 35% が AI 生成 / AI 支援
- AI 生成サイトのポジティブ感情スコアは人間比 +107% (LLM の「ごますり」「過度な楽観主義」の症状)
- 意味類似性スコア +33% (文体でなく考え方そのものの多様性が失われた)

研究 FACT (出典: The Conversation、University at Buffalo / University at New Orleans、北米インディーカフェ調査):
- お気に入りのインディーカフェの約 2/3 が同じ記号 (むき出しレンガ壁 / 黒板メニュー / 再生木材カウンター / 観葉植物 / タトゥーのバリスタ)
- 6 枚の写真で都市判別、全問正解者ゼロ。チェーンを拒否したはずの店が「もう一つのチェーン」化

= 一人ひとりの合理的最適化の総和が、集団としての均質化を生む。AI はその速度と規模を桁違いにする。

スロップは「わかりやすい不良品」でない。表面はきれいで明るく感じがいい、読後に何も残らない。スナック菓子を食べ続けて死ぬ種類の毀損。日々の SNS / バナー / 提案資料 / 内装の「感じのいい」積み重ねが、ブランドの識別性を静かに溶かす。

## 2. ブランド = 「あえてやらなかったこと」の集積

出典: 片石貴展氏 (株式会社ゆとり代表) X ポスト 2026-05-11「合理的に考えたらやった方が良いけど、あえてやってないことの積み重ねがブランドになる」(FACT、ユーザー提示)。

AI は「合理的にやった方が良いこと」(SEO に強い見出し / エンゲージメント取れる構文 / A/B テストで勝つ配色) の最適解出しに人間を圧倒。だがブランドの輪郭は逆側 = 合理的でもあえてやらない判断の積み重ね。

AI 時代ブランディングの本質 = 「AI に何をやらせるか」でなく「AI が合理的に推奨してくる平均値を、どこで断れるか」。

## 3. 断る力 3 能力

| 能力 | 定義 | 制作時チェック項目 |
|---|---|---|
| テイスト | 何を選ばないかの判断力、知識と経験の集積。主文脈に別文脈を当て行間を読む (飲料 × 建築 / 現代美術 / 地方の祭り) | AI 推奨の平均値を 1 点以上意図的に外したか |
| 意思と勇気 | 「何と何を組むか」を決める + 「不安に耐える」組織体質 | 組み合わせ判断を誰が決め、なぜそうしたか記録したか |
| ユーモア | 「わからない人にはわからなくていい」と腹をくくり仲間を選別 (Liquid Death / Oatly / Duolingo) | 全員向けに説明を足して平均値に薄めていないか |

3 能力は AI バージョンが上がってもコモディティ化しない。ツールでなく組織の人間の身体と経験に宿る。

## 4. 識別性ゲート (brand-guardian 定性軸)

YOU MUST: creative 成果物の brand-guardian レビュー時、以下を必須化:
- 「AI / 参考事例が合理的に推奨する案を、どこで断ったか」を 1 行明示。未記載は REVISE
- 機械検出 (ポジティブ感情 +107% / 意味類似性 +33% の自動計測) は採用しない。事後の読者反応指標であり制作時 hook で測れないため。代わりに「断った点の言語化」を採点対象とする
- Goodhart 回避: 逸脱量でなく「断った判断の根拠」を採点 (ただ外せばよいではない)

## 5. 既存品質ゲートとの二層関係

| 層 | 役割 | 実装 |
|---|---|---|
| 床 (スロップ回避) | 最低品質を守る | PR #212 outcomes-judge ブロッキング + template-injection-check / PR #216 ビジュアル品質ルーブリック |
| 天井 (識別性確保) | 平均値からの離脱を促す | 本 skill の識別性ゲート (断る力 3 能力 + 1 行言語化) |

床だけでは「感じのいいスロップ」を量産する。天井ゲートで「あえて断った輪郭」を確保。

## 6. デザインパートナーの役割転換

AI で「感じのいいもの」が無限に手に入る世界では「綺麗なものを作る人」の価値は残らない。残るのは「良薬口に苦し」を理由とセットで処方できる立場。

受託型 (依頼を受けてから動く) は間に合わない。クライアントが「モヤモヤ」を抱えた段階で社内 AI が陽気な「それっぽい案」を出し、議論の土俵が平均値に引き寄せられている。先に具体 (プロトタイプ) を置いた者がブランドの輪郭を引ける。「合理的にはこれだが、あえてこちらでいく」というプロトタイプが議論の土俵を定義する。

## 7. 反証チェック (Step 1-4 圧縮)

- Step 2: PR #212/#216 (床ゲート) + brand-guidelines §6 品質ゲート + creative-playbook と整合、本 skill = 天井ゲートで二層補完
