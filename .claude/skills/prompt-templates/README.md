# プロンプトテンプレ集（30本）

> ConsultingOS 内で頻用するプロンプトの素材集。佐藤裕介・小野寺・主要思想家のフレームを内蔵済み。Khairallah 6要素（役割/文脈/制約/形式/品質基準/具体例）を組み込み済み。

## 使い方

1. 該当カテゴリのプロンプトを選ぶ
2. `[]` のプレースホルダを実際の値で置き換える
3. Claude Code に投入

## 共通原則

全プロンプトに以下を組み込み済み:
- **役割の明示**（「あなたは〜です」）
- **文脈の指定**（[クライアント名][業界][状況]等のプレースホルダ）
- **制約の明文化**（出典必須・断定禁止表現等）
- **出力形式の指定**（テーブル/箇条書き/段階数等）
- **品質基準**（反証チェック必須等）
- **具体例**（Good/Bad パターン）

## 反証モード必須

全プロンプトの末尾に以下を含めて使用すること:
```
最後に以下を必ず付与:
【反証チェック結果】
✅ Step 1（自己反証）
✅ Step 2（構造反証）
✅ Step 3（実用反証）
🔺 残存リスク
```
