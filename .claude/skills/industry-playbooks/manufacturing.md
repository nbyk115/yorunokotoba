# 製造業 業界プレイブック

> ⚠️ **ハルシネーション禁則**: 数値・市場規模等は**断定せず一次ソース確認**。本プレイブックは構造の理解と参照先を提供する。詳細は `.claude/memory/data-sources.md`。

## 1. 業界構造

### 業種分類
- **素材**: 化学・鉄鋼・非鉄金属・セメント・紙パ
- **資本財・機械**: 産業機械・建機・工作機械・電子部品
- **電機・電子**: 半導体・電子部品・家電・産業用ロボット
- **自動車・輸送機器**: 完成車・部品サプライヤー（Tier 1-3）・船舶・航空
- **食品・飲料**: 加工食品・調味料・飲料・酒類
- **生活用品・化学**: 日用品・化粧品・医薬品

### 業界構造の特徴
- **多層サプライチェーン**: 素材→部品→Tier 1→OEM→販売の長いバリューチェーン
- **B2B 主体**: 最終消費者ではなく企業向けが中心（食品・日用品・化粧品は B2C）
- **設備投資・在庫**: 大規模 CAPEX とサプライチェーン管理が経営の中核
- **長期取引**: 5-10年スパンの取引関係（自動車・建機等）
- **品質保証**: ISO 9001 / IATF 16949（自動車）等の認証必須
- **DX 進行**: スマートファクトリー / IoT / 予知保全 / 在庫最適化

- **日経クロステック**: https://xtech.nikkei.com/

## 2. 業界固有 KPI

> ⚠️ 製造業 KPI は業種・事業規模で大きく変動。**断定的ベンチマーク数値の引用禁止**。最新の業界レポート / 同業他社 IR を参照。

### Primary（事業 KPI）
- **売上総利益率（粗利率）**: 製品ミックス・原材料コスト・為替で変動
- **営業利益率**: 製造業のベンチマーク
- **棚卸資産回転率**: 在庫効率
- **総資産回転率**: 資産効率
- **設備投資 / 売上比**（CAPEX intensity）: 業界次第
- **R&D 売上比率**: 技術・電子・化学・医薬で高い

### 製造現場 KPI
- **OEE**（Overall Equipment Effectiveness）: 稼働率 × 性能 × 良品率
- **歩留まり**（Yield）: 生産投入に対する良品率
- **不良率**（Defect Rate / PPM）: 100万個あたりの不良数
- **リードタイム**: 受注→出荷までの期間
- **在庫日数**: 在庫が売上に変わるまでの日数
- **MTBF / MTTR**: 平均故障間隔・平均修復時間

### サプライチェーン KPI
- **OTIF**（On-Time In-Full）: 納期通り・数量通り納品率
- **サプライヤー品質**: PPM 不良率 / 是正処置完了率
- **キャッシュコンバージョンサイクル**: 原材料調達→売掛回収まで

### 参照先
- **日本生産性本部**: 生産性指標
- **日経 業種別経営指標**: 各業種の財務指標
- **日銀短観**: 業況感

## 3. 業界規制・コンプライアンス

### 安全・品質
- **製造物責任法（PL 法）**: 製品事故での無過失責任
- **ISO 9001 / IATF 16949**（自動車）/ AS9100（航空宇宙）/ ISO 13485（医療機器）
- **薬機法**（医薬品・医療機器）: 厚労省 / PMDA 管轄
- **食品衛生法 / 食品表示法**（食品）: 厚労省 / 消費者庁
- **化審法 / 化管法**（化学物質）: 経産省・環境省・厚労省共管

### 環境・脱炭素
- **省エネ法**: エネルギー使用量報告義務
- **温対法（地球温暖化対策推進法）**: GHG 排出量報告
- **CBAM**（EU 国境炭素調整メカニズム）: 2026年〜本格適用、輸出時の炭素価格規制
- **TCFD / ISSB**: 気候関連財務情報開示

### 輸出・通商
- **外為法**: 安全保障輸出管理（リスト規制 / キャッチオール規制）
- **米国輸出管理規則（EAR）/ ITAR**: 米国原産技術の輸出
- **原産地規則**: FTA / EPA 活用での原産地証明

### 労務・安全
- **労働安全衛生法**: 製造現場の安全管理
- **PFAS / RoHS / REACH**（化学物質規制）
- **個人情報保護法**: IoT データ・労働者データ

## 4. 商習慣・契約慣行

### 取引慣行
- **長期相対取引**: 3-5年以上の継続関係（自動車・建機・電子部品等）
- **VMI**（Vendor Managed Inventory）: サプライヤーが顧客在庫を管理
- **JIT**（Just In Time）: 在庫を持たない方式（自動車）
- **Cost Down**: 年次 X% のコストダウン要求が慣行（自動車）
- **下請法**: 下請取引の支払期日・代金減額禁止

### 契約形態
- **基本取引契約 + 個別契約**: MSA + 注文書方式
- **年間契約**: 数量・価格を年間で固定
- **共同開発契約**: NDA + 共同開発契約
- **OEM / ODM 契約**: 委託製造

### 営業プロセス
- **コンペ**: スペック合致後の価格コンペが標準
- **試作・テスト**: 量産前の試作・耐久試験で2-12ヶ月
- **量産承認**: PPAP（Production Part Approval Process）等の承認プロセス
- **長期パートナーシップ**: 一度採用されると数年-数十年継続

## 5. 典型的な失敗パターン

### 失敗 1: サプライチェーン分散の遅れ
- 単一国・単一サプライヤー依存 → 地政学リスク・自然災害で断裂
- → 複数地域・複数社への分散（ニアショア / フレンドショア）

### 失敗 2: 脱炭素対応の遅れ
- CBAM や ESG 投資家圧力で輸出停止・取引停止リスク
- → Scope 1/2/3 の GHG 排出量計測・削減ロードマップ整備

### 失敗 3: スマートファクトリー投資の費用対効果見誤り
- IoT / AI 導入が運用フェーズで活用されない
- → 現場の業務プロセス改善とセットで投資、Day-2 Operations を最初から設計（ハイタワー流）

### 失敗 4: 後継者不足・技能伝承の断絶
- 熟練工の退職で品質低下
- → デジタルツイン / AR 作業支援 / 動画マニュアルでの形式知化

### 失敗 5: 安全保障輸出管理の見落とし
- 米国 EAR / 日本外為法の規制対象品目を無届けで輸出
- → legal-compliance-checker と連携し、輸出前の該非判定を徹底

## 7. 提案・分析時のチェックリスト

### 提案前
- [ ] 業種・サプライチェーン上の位置付け（素材 / 部品 / 完成品）
- [ ] 主要顧客・主要サプライヤーの集中度
- [ ] 規制環境（薬機 / 食品 / 化学物質 / 輸出管理）
- [ ] 脱炭素対応の進捗（Scope 1/2/3 開示状況）

### 提案中
- [ ] OEE / 歩留まり / 不良率の現状値と改善余地
- [ ] サプライチェーン耐性（地政学・自然災害）
- [ ] 投資対効果（CAPEX vs 期待リターン）を PL に落とす
- [ ] 法規制リスク（PL 法 / 外為法 / 環境法）の事前チェック

### 提案後
- [ ] 月次・四半期で OEE / 歩留まり / 不良率の推移モニタリング
- [ ] 規制変更（CBAM / EU AI Act / 化学物質規制）の継続ウォッチ
- [ ] サプライヤー監査の定期実施
